Notre modèle est celui d'une plateforme SaaS : nous nous rémunérons sur l'accès, pas sur les actes de santé. C'est un engagement éthique — aucun conflit d'intérêt sur la qualité ou le volume de plans produits.

SLA garanti : Si un plan n'est pas livré sous 48h ouvrées, le crédit est automatiquement remboursé sur votre compte. Aucune démarche nécessaire.
⚡ Gestion de la file d'attente — ce que vous devez savoir
🔴 Priorité haute (option Urgent)
Vos plans urgents passent devant les patients directs et prescripteurs. Utilisez-le pour les membres avec pathologie ou échéance courte. SLA réduit à 24h ouvrées.
📦 Commandes groupées (batch)
Si vous commandez plus de 10 plans en une fois, ils sont automatiquement répartis sur plusieurs diét. simultanément pour tenir le délai 48h. Pas besoin de les étaler manuellement.
⚠️ Quota mensuel
Votre offre inclut un plafond mensuel (visible dans vos KPIs). Si vous approchez 80%, une alerte apparaît automatiquement. Contactez-nous pour augmenter votre quota.
🔌 Automatisation via API
Les offres Volume et Grands comptes ont accès à l'API REST pour déclencher des commandes directement depuis votre logiciel (CRM, logiciel salle, Doctolib…). Contactez-nous pour les clés.
